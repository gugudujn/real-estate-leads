# AI Rules for Real Estate Lead App

## Core Principles
- Do not break webhook integration
- Keep UI simple and clean
- Do not over-engineer

## UI/UX
- Mobile responsive first
- Minimal and fast experience
- No unnecessary animations

## Data Handling
- Always preserve existing lead fields
- Do not remove fields used in webhook
- Keep data structure consistent

## Coding Style
- Keep components readable
- Avoid unnecessary abstraction
- Prefer simple React patterns

## Business Logic
- Prioritize lead capture
- Do not reduce conversion elements
- Keep assistant conversational and clear
